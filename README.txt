Astroids retro game made from August 12 - 17.

Steps to install:

1: Put all the files in a folder
3: Make sure you have python installed, if not head to www.python.org
4: Make sure you have pip installed, if not go to pip.pypa.io/en/stable/installation
5: Run in your command prompt: 'pip install pyxel'
6: Go to the Folder in your command prompt and type: 'pyxel run Asteroids.py'
